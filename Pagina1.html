Product Specification

  • Cursul de WEB Development este un curs special conceput pentru a asigura o insertie profesională rapidă a absolventilor săi. Un absolvent al cursului va putea să se angajeze pe poziții care cer cunostințe de HTML, CSS, JavaScript.Cursul deschide perspective și pentru aceia care doresc sa devina freelanceri și să dezvolte aplicații web pe bază de contracte.
  • HTML, CSS si JavaScript vor fi doar niste instrumente de lucru care ne vor ajuta sa gândim interfața noastră web. Scopul cursului este să vă ajutăm sa gandiți interfețe web prietenoase.
  • Vom studia HTML5, CSS3 si JavaScript. Pentru a crea animații și efecte speciale vom studia si JQuery.
  • Vom studia si Bootstrap pentru a crea astfel de site-uri responsive. Cursul se va finaliza cu elemente de marketing digital pentru a promova site-ul pe care l-ati realizat.
  • Este un program full-stack care vă învață cum să dezvoltați și partea de backend a unei aplicații web utilizând NodeJS și baza de date MySQL.
  • Pentru cei care acum sunteți la primul contact cu "liniile de cod" vă recomandăm cu plăcere acest curs.
