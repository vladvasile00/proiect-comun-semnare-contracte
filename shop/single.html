Product Specification

  • Cursul de JAVA oferit de IT LABS este un curs în format unic, atât la nivel de conținut cât și ca abordare.
  • Acest curs este dedicat atât persoanelor care nu au studiat nicioadată programarea.
  • Cursul nostru se va axa pe limbajul JAVA 9 și va prezenta aspectele nou introduse în limbaj precum expresiile lambda, stream-uri, Java Shell, fabricile pentru containere, etc
  • Avem în vedere bineînțeles să vă prezentăm conceptele clasice de programare dar și cele moderne, introduse în ultima versiune de JAVA lansată pe piată în toamna anului 2017.
  • Dacă ați studiat o facultate în domeniu, sau până acum ați programat în alte limbaje de programare acest curs este o oportunitate foarte bună pentru a vă actualiza cunostințele despre JAVA 9.
  • Facultățile din țară înca predau Java 6, insă IT LABS vă oferă posibilitatea de a fi la curent cu ultimele actualizări asupra limbajului.
  • Pentru cei care acum sunteți la primul contact cu "liniile de cod" vă recomandăm cu plăcere acest curs. Vom avea grijă ca fiecare detaliu să fie înțeles.
